Welcome to the Movie Recommendation System

Select a movie from the dropdown list to get recommendations based on user ratings.

The similarity score indicates how similar two movies are based on user ratings. It ranges from 0 to 1, where 0 indicates no similarity and 1 indicates perfect similarity.

When users rate movies, they express their preferences. By comparing these ratings, we can find movies that share similar patterns of user preference. For example, if many users rate both "Movie A" and "Movie B" highly, these movies are considered similar and "Movie B" can be recommended to users who liked "Movie A".

Example:

User Movie A Movie B
1 5 4
2 4 5
3 1 2
4 5 5
5 3 3

The similarity score between Movie A and Movie B would be high because their ratings are similar across multiple users. This implies that users who liked Movie A are also likely to like Movie B.
